# Personal Website Starter

A blank HTML/CSS personal website template, ready to build on.

## Stack
- Plain HTML (`index.html`) and CSS (`style.css`) — no build step, no dependencies.

## How to run
The workflow `Start application` serves the site with Python's built-in HTTP server:
```
python3 -m http.server 5000
```
The preview pane shows the site at port 5000.

## How to develop
Edit `index.html` and `style.css` directly. Changes appear immediately on page refresh — no restart needed.
